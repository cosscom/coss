# @coss/calendar

Coss calendar simplifies the interaction with third party calendars.

## Installation

```bash
npm install @coss/calendar
# or
yarn add @coss/calendar
# or
pnpm add @coss/calendar
# or
bun add @coss/calendar
```

## Usage

```typescript
import { something } from "@coss/calendar";

console.log(something);
```
