# coss Component Registry Index

Use this file to quickly identify the right coss primitive for a UI task. Each entry includes the component name, a one-line purpose, and the skill path for detailed guidance.

## Overlays & Popups
- **Dialog** — Centered modal requiring user focus. `primitives/coss-dialog/`
- **AlertDialog** — Destructive/critical confirmation modal. `primitives/coss-alert-dialog/`
- **Sheet** — Side-panel overlay for settings/details. `primitives/coss-sheet/`
- **Drawer** — Bottom/side drawer, often mobile-responsive. `primitives/coss-drawer/`
- **Popover** — Anchored non-modal floating content. `primitives/coss-popover/`
- **Tooltip** — Short hover/focus hint text. `primitives/coss-tooltip/`
- **PreviewCard** — Hover-triggered rich entity preview. `primitives/coss-preview-card/`
- **Menu** — Dropdown action list with groups/submenus. `primitives/coss-menu/`
- **Command** — Searchable command palette (not cmdk). `primitives/coss-command/`

## Selection & Input
- **Select** — Single-choice from predefined list (no search). `primitives/coss-select/`
- **Combobox** — Searchable selection with filtering. `primitives/coss-combobox/`
- **Autocomplete** — Free-text with suggestions. `primitives/coss-autocomplete/`
- **Input** — Single-line text entry. `primitives/coss-input/`
- **Textarea** — Multi-line text entry. `primitives/coss-textarea/`
- **InputGroup** — Input with addons (icons, buttons, badges). `primitives/coss-input-group/`
- **InputOTP** — One-time passcode segmented slots. `primitives/coss-input-otp/`
- **NumberField** — Numeric entry with stepper controls. `primitives/coss-number-field/`
- **Slider** — Continuous/ranged numeric control. `primitives/coss-slider/`
- **Calendar** — Date picker / calendar views. `primitives/coss-calendar/`

## Forms & Validation
- **Form** — Form validation/submission with Zod. `primitives/coss-form/`
- **Field** — Label + description + error wiring. `primitives/coss-field/`
- **Fieldset** — Grouped form controls with legend. `primitives/coss-fieldset/`
- **Label** — Accessible label for controls. `primitives/coss-label/`

## Toggle & Choice
- **Checkbox** — Single boolean toggle. `primitives/coss-checkbox/`
- **CheckboxGroup** — Multiple-selection set. `primitives/coss-checkbox-group/`
- **RadioGroup** — Mutually exclusive single choice. `primitives/coss-radio-group/`
- **Switch** — Binary on/off preference toggle. `primitives/coss-switch/`
- **Toggle** — Pressable two-state command button. `primitives/coss-toggle/`
- **ToggleGroup** — Grouped pressed-state controls. `primitives/coss-toggle-group/`

## Layout & Navigation
- **Tabs** — Mutually exclusive tabbed panels. `primitives/coss-tabs/`
- **Accordion** — Collapsible content sections. `primitives/coss-accordion/`
- **Collapsible** — Single expand/collapse region. `primitives/coss-collapsible/`
- **Sidebar** — Persistent app shell navigation. `primitives/coss-sidebar/`
- **Breadcrumb** — Hierarchical navigation trail. `primitives/coss-breadcrumb/`
- **Pagination** — Paged navigation controls. `primitives/coss-pagination/`
- **Toolbar** — Grouped command/action strip. `primitives/coss-toolbar/`
- **ScrollArea** — Styled scroll container. `primitives/coss-scroll-area/`

## Content & Display
- **Card** — Content container with sections. `primitives/coss-card/`
- **Frame** — Bordered content surface. `primitives/coss-frame/`
- **Table** — Tabular data presentation. `primitives/coss-table/`
- **Avatar** — User/entity profile image. `primitives/coss-avatar/`
- **Badge** — Status indicator / label. `primitives/coss-badge/`
- **Kbd** — Keyboard shortcut hints. `primitives/coss-kbd/`
- **Separator** — Visual/semantic divider. `primitives/coss-separator/`
- **Group** — Connected control cluster. `primitives/coss-group/`
- **Empty** — Empty-state placeholder. `primitives/coss-empty/`

## Feedback & Status
- **Alert** — Inline persistent status message. `primitives/coss-alert/`
- **Toast** — Transient notification (toastManager). `primitives/coss-toast/`
- **Progress** — Task completion / async progress bar. `primitives/coss-progress/`
- **Meter** — Bounded scalar measurement. `primitives/coss-meter/`
- **Spinner** — Indeterminate loading indicator. `primitives/coss-spinner/`
- **Skeleton** — Loading placeholder. `primitives/coss-skeleton/`

## Actions
- **Button** — Primary/secondary action trigger. `primitives/coss-button/`
