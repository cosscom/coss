# coss Component Registry Index

Use this file to quickly identify the right coss primitive for a UI task. Each entry includes the component name, a one-line purpose, and the reference path for detailed guidance.

## Overlays & Popups
- **Dialog** — Centered modal requiring user focus. `primitives/dialog.md`
- **AlertDialog** — Destructive/critical confirmation modal. `primitives/alert-dialog.md`
- **Sheet** — Side-panel overlay for settings/details. `primitives/sheet.md`
- **Drawer** — Bottom/side drawer, often mobile-responsive. `primitives/drawer.md`
- **Popover** — Anchored non-modal floating content. `primitives/popover.md`
- **Tooltip** — Short hover/focus hint text. `primitives/tooltip.md`
- **PreviewCard** — Hover-triggered rich entity preview. `primitives/preview-card.md`
- **Menu** — Dropdown action list with groups/submenus. `primitives/menu.md`
- **Command** — Searchable command palette (not cmdk). `primitives/command.md`

## Selection & Input
- **Select** — Single-choice from predefined list (no search). `primitives/select.md`
- **Combobox** — Searchable selection with filtering. `primitives/combobox.md`
- **Autocomplete** — Free-text with suggestions. `primitives/autocomplete.md`
- **Input** — Single-line text entry. `primitives/input.md`
- **Textarea** — Multi-line text entry. `primitives/textarea.md`
- **InputGroup** — Input with addons (icons, buttons, badges). `primitives/input-group.md`
- **InputOTP** — One-time passcode segmented slots. `primitives/input-otp.md`
- **NumberField** — Numeric entry with stepper controls. `primitives/number-field.md`
- **Slider** — Continuous/ranged numeric control. `primitives/slider.md`
- **Calendar** — Date picker / calendar views. `primitives/calendar.md`

## Forms & Validation
- **Form** — Form validation/submission with Zod. `primitives/form.md`
- **Field** — Label + description + error wiring. `primitives/field.md`
- **Fieldset** — Grouped form controls with legend. `primitives/fieldset.md`
- **Label** — Accessible label for controls. `primitives/label.md`

## Toggle & Choice
- **Checkbox** — Single boolean toggle. `primitives/checkbox.md`
- **CheckboxGroup** — Multiple-selection set. `primitives/checkbox-group.md`
- **RadioGroup** — Mutually exclusive single choice. `primitives/radio-group.md`
- **Switch** — Binary on/off preference toggle. `primitives/switch.md`
- **Toggle** — Pressable two-state command button. `primitives/toggle.md`
- **ToggleGroup** — Grouped pressed-state controls. `primitives/toggle-group.md`

## Layout & Navigation
- **Tabs** — Mutually exclusive tabbed panels. `primitives/tabs.md`
- **Accordion** — Collapsible content sections. `primitives/accordion.md`
- **Collapsible** — Single expand/collapse region. `primitives/collapsible.md`
- **Sidebar** — Persistent app shell navigation. `primitives/sidebar.md`
- **Breadcrumb** — Hierarchical navigation trail. `primitives/breadcrumb.md`
- **Pagination** — Paged navigation controls. `primitives/pagination.md`
- **Toolbar** — Grouped command/action strip. `primitives/toolbar.md`
- **ScrollArea** — Styled scroll container. `primitives/scroll-area.md`

## Content & Display
- **Card** — Content container with sections. `primitives/card.md`
- **Frame** — Bordered content surface. `primitives/frame.md`
- **Table** — Tabular data presentation. `primitives/table.md`
- **Avatar** — User/entity profile image. `primitives/avatar.md`
- **Badge** — Status indicator / label. `primitives/badge.md`
- **Kbd** — Keyboard shortcut hints. `primitives/kbd.md`
- **Separator** — Visual/semantic divider. `primitives/separator.md`
- **Group** — Connected control cluster. `primitives/group.md`
- **Empty** — Empty-state placeholder. `primitives/empty.md`

## Feedback & Status
- **Alert** — Inline persistent status message. `primitives/alert.md`
- **Toast** — Transient notification (toastManager). `primitives/toast.md`
- **Progress** — Task completion / async progress bar. `primitives/progress.md`
- **Meter** — Bounded scalar measurement. `primitives/meter.md`
- **Spinner** — Indeterminate loading indicator. `primitives/spinner.md`
- **Skeleton** — Loading placeholder. `primitives/skeleton.md`

## Actions
- **Button** — Primary/secondary action trigger. `primitives/button.md`
